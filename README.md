# Ikseer Al Haya Trading — Official Website

**ikseeralhaya.com** — Bilingual (English / Arabic) single-page website.

## Sections
- **Hero** — headline, CTA, animated convergence mark, stats
- **About** — company story, 4 activities panel, 10-year badge
- **Services** — 4 licensed activity cards (01–04)
- **Why Choose Us** — 5 reasons with brand-gradient checks
- **Contact** — dark panel with phone, email, website + CTA buttons
- **Footer** — brand mark, nav links, contact info, activities list

## Features
- ✅ Bilingual EN ↔ AR with RTL support (click the globe button)
- ✅ Language preference saved in localStorage
- ✅ Fully responsive — desktop, tablet, mobile
- ✅ Sticky, blur-backdrop navigation
- ✅ Animated brand mark (three colored orbs = company logo colours)
- ✅ No build step — pure HTML/CSS/JS
- ✅ All fonts via Google Fonts (Cairo + Tajawal)

## File Structure
```
index.html          ← main page
assets/
  styles.css        ← all styles
  script.js         ← language toggle + mobile nav
  icon.png          ← logo icon (nav + favicon)
  logo-full.png     ← full logo with text
```

## Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `ikseeralhaya-website`)
2. Upload all files **maintaining the folder structure above**
3. Go to **Settings → Pages → Source: Deploy from branch → main / (root)**
4. Save — your site will be live at `https://USERNAME.github.io/REPO/`
5. (Optional) Add a custom domain `ikseeralhaya.com` in the Pages settings
